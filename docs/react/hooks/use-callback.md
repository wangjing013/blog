# useCallback

``useCallback`` 是 React 中的一个钩子函数，它用于返回一个 ``memoized（记忆化）`` 的回调函数。在使用时，需要显式地指定依赖项数组，其目的是控制回调函数在何时需要重新创建。通过这种机制，可以有效避免组件在每次重新渲染时都去创建新的函数，从而提升渲染性能。当我们将回调函数传入到子组件中时，这可能非常有用。

> ``什么是 memoized 函数？``: 函数记忆是指将上次的计算结果缓存起来，当下次调用时，如果遇到相同的参数，就直接返回缓存中的数据。 原理：实现这样一个 ``memoize`` 函数很简单，原理上只用把参数和对应的结果数据存到一个对象中，调用时，判断参数对应的数据是否存在，存在就返回对应的结果数据。

> 当我们使用 ``React Compiler`` 时，系统会自动对值和函数进行 ``memoized`` 处理。这样可以减少手动调用 ``useCallback`` 的需要。
