# V2rayNG

V2rayNG 是一款基于 V2Ray 核心的 Android 端网络代理工具，提供了丰富的功能和灵活的配置选项，适用于科学上网、隐私保护和网络加速等多种场景。





## 路由规则详解

### 一、 域名策略 (Domain Strategy)

核心逻辑：决定如何处理网址到 IP 的转换过程。
* AsIs (保持原样): 仅根据域名进行匹配。如果规则里没有这个域名，就直接交给服务器处理，不会尝试解析 IP。（不建议用于换 IP 场景，容易漏掉基于 IP 段的规则）。
* IPIfNonMatch (域名不匹配时解析 IP): 最推荐的模式。先匹配域名，没中，就解析成 IP 再匹配一遍。（最平衡，换 IP 必备）。
* IPOnDemand (按需解析 IP): 只要规则里涉及到 IP 判断，立即进行解析。（最严谨，但如果 DNS 不稳可能会有轻微延迟）。

### 二、 基础功能开关

* 阻断 UDP443 (QUIC): 拦截采用 QUIC 协议（如 YouTube/Google）的 UDP 流量，强制其走 TCP 代理。这能解决部分代理服务器 UDP 不通导致的连接超时。
* 代理 Google: 强制将 Google 相关服务划入代理名单。

### 三、 绕过规则（换 IP 时的“红区”）

注意： 如果你的目标是从 A 城市换到 B 城市，以下所有“绕过中国”的选项都必须关闭 (OFF)。

* 绕过中国 IP [geoip:cn]: * 
    * 开启: 访问国内 IP 时用本地原 IP（直连）。
    * 关闭: 访问国内 IP 时强制走代理节点（换 IP 成功）。
* 绕过中国域名 [geosite:cn]: 
    * 开启: 识别到 .cn 或国内主流域名时直连。
    * 关闭: 强制国内域名也通过代理服务器加载。
* 绕过中国 DNS/DNS IP: 控制解析过程是否走本地，在换 IP 场景，全部关闭。虽然会导致解析稍慢，但能有效防止“DNS 泄漏”（即对方通过 DNS 发现你真实的物理城市）。

### 四、 局域网规则（建议保持开启）

绕过局域网域名/IP: 是指在访问局域网内的IP/域名地址时不使用代理服务器。
